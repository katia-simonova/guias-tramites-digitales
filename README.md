# Guías de trámites digitales

Guías claras paso a paso para realizar trámites digitales desde el móvil.

Este repositorio contiene instrucciones prácticas para ayudar a personas usuarias a completar trámites online con menos confusión, especialmente cuando no disponen de ordenador.

## Objetivo

Transformar procesos administrativos complejos en guías claras, ordenadas y fáciles de seguir.

## Formato de cada guía

Cada guía debe incluir:

- Nombre del trámite
- Para qué sirve
- Quién puede hacerlo
- Requisitos previos
- Documentos necesarios
- Pasos desde el móvil
- Errores frecuentes
- Qué hacer si el trámite falla
- Resultado esperado
- Próxima acción

## Temas previstos

- Inscripción a cursos
- Cita previa
- Renovación de demanda de empleo
- Envío de documentación
- Solicitud de certificados
- Uso de formularios online
- Organización de documentos PDF

## Principios de redacción

- Lenguaje claro
- Pasos cortos
- Orden lógico
- Sin tecnicismos innecesarios
- Explicaciones útiles para personas migrantes o con baja competencia digital
- Enfoque mobile-first
