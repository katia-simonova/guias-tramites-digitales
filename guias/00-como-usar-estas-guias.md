# Cómo usar estas guías

Estas guías están pensadas para personas que realizan trámites digitales principalmente desde el móvil.

## Principios

- Pasos cortos
- Lenguaje claro
- Orden lógico
- Sin datos personales
- Verificación con fuentes oficiales
- Atención a errores frecuentes

## Antes de seguir una guía

Comprobar siempre:
- que la página sea oficial
- que la información esté actualizada
- que los documentos no contengan errores
- que se guarde justificante si existe

## Seguridad

No subir a repositorios públicos:
- DNI/NIE/pasaporte
- direcciones privadas
- datos médicos
- datos bancarios
- números de expediente
- capturas con información personal
